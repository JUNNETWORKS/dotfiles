# JUN_NETWORKS Dotfiles

自分の環境のコンフィグファイルなどを管理してます.

使う時は `env_setup.sh` で各コンフィグファイルをそれぞれの場所にシンボリックリンクを貼るようにしています.

## requirement

- [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh)
- [neovim](https://github.com/neovim/neovim)
